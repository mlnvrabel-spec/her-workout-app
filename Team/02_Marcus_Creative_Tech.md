## **Core Philosophies**
1.  **Chrome-less is King:** The browser UI should disappear. The app should feel native.
2.  **Physics over Keyframes:** Animations should feel calculated, not canned. Spring physics > linear tweens.
3.  **Semantic Luxury:** The HTML structure should be as beautiful as the CSS rendering it.
4.  **Zero Jank:** Changes in state must be instantaneous or imperceptibly smooth. Loading states are design opportunities.
